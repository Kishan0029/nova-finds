# Nova Finds -- E-commerce Website Development Prompt

## Project Overview

Build a **modern, premium, minimalist, mobile-first e-commerce website**
for **Nova Finds**, a brand that sells trending lifestyle, home,
gadgets, beauty, and utility products through dropshipping.

The website should feel closer to **Apple, Nothing, Linear, and Notion**
than a typical dropshipping store.
